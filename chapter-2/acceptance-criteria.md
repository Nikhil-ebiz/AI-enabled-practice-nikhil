## Acceptance Criteria

- User should have an option to snooze a particular email thread on hovering over the email thread.
- User should see the option to snooze email threads on options menu when user selects the email threads.
- By clicking on the snooze button, a modal should open with option to select day of the week and time window for that day of the week.
- User should not get the notification of the snoozed email threads in scheduled focus time.
- Unsnoozed email thread's notification should be unaffected.
- User can have single email account logged to multiple devices, and none of them should get notification if notifications are snoozed per that device's timezone.
